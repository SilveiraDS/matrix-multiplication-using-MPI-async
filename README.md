Trabalho 3_a da Disciplina de IPPD - CMP134

Dieison Silveira

Produto de matrizes NxN com MPI ponto a ponto assíncrono (usar as operações MPI_Isend, MPI_Irecv, MPI_Test, MPI_Wait e variantes)

Compilar:
$ cd dieison
$ make

Compilar com scorep:
$ cd dieison
$ make scorep

Executar:
$ mpirun -np <#nodos> ./main <#size>
